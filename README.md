== README

This is a 2-sided marketplace created following the course at [Codeplace.com](https://www.codeplace.com/stuk-io/2-sided-marketplace/).

Goals covered in this course:

* Payments
* State Machines
* Stripe SDK
* Environment Variables
* Friendly URLs
* Amazon S3 SDK
* Authorisations
* File Attachments
* Validations
